# RealTime Chat Application

A professional real-time chat application with PWA support, built with Node.js, Socket.io, and MySQL.

## Features

- ✅ Real-time messaging with Socket.io
- ✅ Multiple chat rooms
- ✅ File and image sharing
- ✅ Emoji support
- ✅ Typing indicators
- ✅ Online status
- ✅ Message search
- ✅ PWA (Progressive Web App)
- ✅ Responsive design
- ✅ User authentication

## Quick Start

### Local Development
```bash
# Clone repository
git clone https://github.com/yourusername/realtime-chat-app.git
cd realtime-chat-app

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your database credentials

# Start development server
npm run dev
